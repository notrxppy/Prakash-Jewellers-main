# Prakash Jewellers – Shopping Cart App

## Overview

* Android jewellery shopping app built using Kotlin and Jetpack Compose
* Allows users to browse products and view details
* Simulates a basic shopping and payment experience

## Features

* Browse jewellery items (Gold, Silver, Diamond)
* View product details with images
* Basic shopping cart functionality
* QR-based payment simulation

## Tech Stack

* Kotlin
* Jetpack Compose
* Android Studio

## Project Structure

* MainActivity.kt – Entry point
* model/JewelryItem.kt – Product data model
* ui/theme – UI design and styling
* DetailScreen.kt – Product detail screen
* res/drawable – Images and assets

## App Flow

* Open app
* Browse products
* Select item
* View details
* Proceed to payment
